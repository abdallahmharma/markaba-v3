# Routes

## Public

| Route | Description |
|-------|-------------|
| `/` | Homepage |
| `/manufacturers` | Manufacturers listing |
| `/models` | Models listing |
| `/vehicles` | Vehicles listing |
| `/vehicle/:id` | Vehicle detail page |
| `/login` | User login |
| `/register` | User registration |
| `/forgot-password` | Password recovery |

## Private

| Route | Description |
|-------|-------------|
| `/profile` | User profile management |
| `/favorites` | User's saved vehicles |
| `/compare` | Vehicle comparison tool |

## Admin

| Route | Description |
|-------|-------------|
| `/admin` | Admin dashboard |
| `/admin/manufacturers` | Manage manufacturers |
| `/admin/models` | Manage models |
| `/admin/vehicles` | Manage vehicles |
| `/admin/users` | Manage users |
| `/admin/reviews` | Manage reviews |
